Predicting Carbon Monoxide Emissions from Motor Vehicles

Project Overview:
This project employed regression and classification machine learning algorithms to predict carbon monoxide emissions from motor vehicles and classify them based on fuel consumption. The goal was to develop models capable of predicting emissions accurately, classifying vehicles, and grouping them based on features.

Data Overview and Cleaning

The dataset includes vehicle emissions and relevant features. The data underwent several cleaning steps to ensure quality and consistency:

  • Data Source: The dataset was sourced from the Canadian government portal: Fuel Consumption Rating Dataset.
  • Data Cleaning: Missing values were handled, and numeric and categorical variables were separated for processing.
  • Feature Encoding: Categorical variables were encoded, and numerical features were scaled using normalization techniques.

Exploratory Data Analysis (EDA)

EDA was performed to understand trends and relationships in the dataset. Key findings included:

  • Vehicle Features: Key features influencing CO2 emissions included Engine Size and Fuel Consumption.
  • Fuel Consumption and Emissions: A correlation was found between fuel consumption and CO2 emissions across different vehicle models.

Machine Learning Models

The project applied both regression and classification models to predict emissions and classify vehicles:

  • Regression: Multiple linear and decision tree regression models were cross-validated using rÂ² and mean squared error (MSE). The decision tree model with features such as Engine Size and Fuel Consumption (mpg) performed best, showing little improvement in CO2 emissions from 2010 to 2014.
  • Classification: The decision tree classifier achieved the best performance, with Fuel Type correctly classifying 96% of the data.
  • Overfitting Check: Overfitting was assessed by comparing training and validation set performances. Cross-validation was used to ensure model generalizability.

Clustering

KMeans clustering was applied to the numeric data, with the following insights:

  • Cluster Analysis: "Fuel Type" and "Model" were identified as the best descriptors for the formed groups.
  • Evaluation Metrics: The Davies-Bouldin and Normalized Mutual Information Score metrics were used to evaluate clustering performance.

Key Findings

  • Accurate Prediction: The machine learning models demonstrated high accuracy in predicting CO2 emissions.
  • Classification Accuracy: Fuel Type classification achieved 96% accuracy using decision tree classifiers.
  • Impact of Vehicle Features: Key features such as Engine Size and Fuel Consumption play a significant role in emissions prediction.

Conclusion

This project successfully demonstrated the application of machine learning techniques for predicting carbon monoxide emissions from vehicles and classifying them based on fuel consumption. The models achieved high accuracy, making them valuable tools for environmental monitoring. However, improvements in CO2 emissions from 2010 to 2014 were minimal, suggesting that more advanced techniques or additional data may be needed for further improvement in emissions prediction.
